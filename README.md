# KieBot
KieBot adalah bot Discord modular yang dibangun khusus untuk komunitas Torn City. Dirancang agar scalable dan mudah dikembangkan, KieBot mendukung **command hybrid** (awalan `!` dan garis miring `/`), serta terintegrasi langsung dengan **Torn City API**. Fitur utamanya mencakup sistem undian (Lotto) yang lengkap, perintah kontrol ID pemain, logging interaktif, serta sistem izin yang fleksibel. Dengan arsitektur berbasis folder yang terstruktur dan dukungan penuh ESM, KieBot siap menjadi fondasi bot all-in-one sekelas MEE6 atau CarlBot â€” tapi khusus untuk dunia Torn. Proyek ini masih dalam tahap pengembangan aktif dan terus dikembangkan dengan fitur-fitur baru yang relevan untuk komunitas game dan Discord modern.

## Features :
- [x] Hybrid command (! dan /)
- [x] Integrasi Torn City API
- [x] Command Lotto lengkap
- [x] Sistem permission & logging
- [x] Auto deploy slash command
- [x] Modular folder structure
